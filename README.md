# Chinook SQL Analysis

## Project Overview

This project analyzes the Chinook music database using SQL and SQLite to uncover insights about customer behavior, sales performance, music trends, and employee workload.

The goal of this project was to demonstrate how relational database queries can be used to transform raw data into meaningful business insights.

## Dataset

The project uses the Chinook SQLite database, a sample database that represents a digital music store. The database contains information about:

- Customers
- Employees
- Artists
- Albums
- Tracks
- Genres
- Invoices
- Sales transactions

Dataset source:
Chinook Database (SQLite sample database)

## Analysis Performed

This project includes:

### Customer Analysis
- Customer distribution by country
- Top customers by total spending

### Sales Analysis
- Monthly revenue trends
- Revenue performance over time

### Music Analysis
- Revenue by genre
- Top revenue-generating artists

### Employee Analysis
- Customer assignments by support representative

## SQL Skills Demonstrated

- SELECT statements
- Filtering and sorting data
- Aggregate functions (SUM, COUNT)
- GROUP BY
- ORDER BY
- SQL JOIN operations
- Multi-table relational analysis
- Revenue calculations

## Key Findings

- The United States had the largest customer base in the database.
- High-value customers were distributed across multiple countries.
- Monthly revenue remained relatively consistent with occasional increases.
- Rock generated the highest revenue among music genres.
- Iron Maiden was the top revenue-generating artist.
- Customer assignments were evenly distributed among support representatives.
